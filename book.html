Plan Your Aerial Shoot With Confidence

Booking starts with a planning request rather than confirmed scheduling or automatic dispatch. Share enough context to review the project without sending private property or customer details in the first message.

Helpful first message

  • Project type and creative goal.
  • General town, venue type, or property context.
  • Preferred timing and final use.
  • Photo, video, or mixed deliverable needs.

Keep out at first

  • Private addresses unless requested later.
  • Personal customer or attendee information.
  • Documents you are not authorized to share.
  • Claims that a project is already approved for capture.

What happens next

  • The request is reviewed for project fit.
  • Relevant portfolio styles can be compared.
  • Questions and constraints are clarified.
  • Scope can be discussed only after review.

What is not included here

  • No live transaction or scheduling system.
  • No automatic pilot assignment.
  • No guarantee of capture feasibility.
  • No approval, credential, coverage, or operating claims.

Next step

Start with a careful project request.

Email the project type, intended use, timing, and general context so the request can be reviewed before details are discussed.
